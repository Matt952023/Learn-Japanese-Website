# Learn-Japanese-Website
An online learning platform for learning the Japanese hirigana and katana alphabets.
- Basic Features: List of lessons with introductory material and display of basic Hiragana and Katakana alphabets.
- Enhanced Features: Simple 10 Question quizzes for basic Hiragana and Katakana characters.
- Advanced Features: An AI Tutor that will help answer user questions. Powered through a backend server utilizing Node.js and OpenAI's API.

Note: To use the AI Tutor, use a .env file with an OpenAI API Key.
